Yolo DeepSort

  • Step1 - Installation of Anaconda
    • Download with this link(Windows), after that you will find the installer exe file click it and setup
    • Change to Destination folder → D:\anaconda3. Then click next until end to finished the installation.
  • Step 2 - Setup Tesseract OCR
    • Download Tesseract OCR with link
    • Install and change the destination folder → C:\Program Files\Tesseract-OCR
    • Click install to finish the installation
    • Copy path to environment path variable → C:\Program Files\Tesseract-OCR\
  • Step 3 - Setup Git
    • Download the Git installer here
    • Click the installer and click next button until the end to install the Git
  • Step 4 - Set up Anaconda environments
    • Open Anaconda Prompt
    • Enter the command to initialize conda environment in powershell → conda init powershell
    • Enter the command to create a new environment → conda create --name Tracking python=3.9
    • Download ultralytics site-package here
    • Extract the download file
    • Click in the extracted folder and copy both file into this path → D:\anaconda3\envs\Tracking\Lib\site-packages

  • Step 5 - Prepare Source
    • Download yolo_tracking zip file in this link and extract it
    • Open source folder in the extracted yolo_tracking folder. Two file will inside source folder.
    • Copy video in to source folder and rename into source.mp4
    • Right click and click open in terminal
    • Enter command in terminal to activate environment by typing → conda activate tracking
    • Generate the video screenshot photo (out.jpg) for label by enter command → python .\screenshot.py
    • Open labelImag tool to label the inside and outside region by enter command → python ..\labelImg\labelImg.py
    • Open (out.jpg) generated source video screenshot picture
    • Change the save directory to the source folder in yolo_tracking
    • Press ‘w’ key to label the image and chose the correct label name
    • Click to save the file and you can find the out.xml file in the source folder
    • Open terminal again to convert xml file format to txt format by enter command → python .\singleXML2yolo.py
  • Step 6 - Run the video
    • Open terminal in Source folder
    • Type command to switch to tracking environment → conda activate tracking
    • Type command to run the video → python ../examples/track.py --show
    • The output file(out.xlsx) is generated in source folder
